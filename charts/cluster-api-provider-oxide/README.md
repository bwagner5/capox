# cluster-api-provider-oxide

The Cluster API Provider for Oxide (CAPOx)

## Installing the Chart

To install the chart with the release name `capox`:

```console
helm upgrade --install capox oci://ghcr.io/oxidecomputer/cluster-api-provider-oxide/helm-charts/cluster-api-provider-oxide
```
